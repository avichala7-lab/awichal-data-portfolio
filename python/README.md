# Python Projects

This directory contains Python utilities and experiments used across the portfolio.

## Tools
- **simple_calculator.py**: Command-line calculator for quick addition, subtraction, multiplication, and division tasks.

### Usage
Run any operation by providing the operation name followed by numbers:

```bash
python simple_calculator.py add 2 3
python simple_calculator.py subtract 10 4 1
python simple_calculator.py multiply 3 5 2
python simple_calculator.py divide 20 4
```
